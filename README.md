# 💖 DataDash: Analytics Suite & Project Builder

DataDash is a high-fidelity, full-stack analytics dashboard featuring a dynamic \"Baby Pink\" theme and a powerful drag-and-drop layout builder. This project demonstrates a sophisticated integration of React, Tailwind CSS, and Node.js to create a premium, data-driven user experience.

## 🚀 Key Features

- **🎛️ Dynamic Dashboard Builder**: A fully interactive interface allowing users to add, resize, and rearrange analytics widgets (KPIs, Charts, Tables) with real-time grid snapping using \eact-grid-layout\.
- **📊 Advanced Data Visualization**: Custom-styled interactive charts—including Area, Bar, Line, Scatter, and Pie—powered by \echarts\, with bespoke tooltips and thematic rose-colored accents.
- **🎨 Bespoke \"Baby Pink\" UI**: A premium, \"lite\" theme meticulously crafted with **Tailwind CSS**. It features glassmorphism, smooth micro-animations, and a curated rose-to-purple gradient system.
- **📦 Order Management System**: A complete CRUD interface for managing transactions, featuring complex filtering, searching, and status tracking.
- **🛡️ Full-Stack Robustness**: A Node.js/Express backend ensures persistent state for dashboard configurations and transaction data, with optimized API endpoints.
- **📱 Ultra-Responsive Design**: A fluid layout system that scales seamlessly from mobile top bars to full-screen desktop grids.

## 🛠️ Technical Stack

- **Frontend**: React 18, Tailwind CSS, Recharts, React Hook Form, React Grid Layout.
- **Backend**: Node.js, Express, MongoDB/Mongoose.
- **Tooling**: Vite, PostCSS, Lucide Icons.

## 🏁 Quick Start

### 1. Clone the repository
\\\ash
git clone https://github.com/yourusername/datadash.git
cd datadash
\\\

### 2. Install Dependencies
\\\ash
# Install backend deps
cd backend && npm install

# Install frontend deps
cd ../frontend && npm install
\\\

### 3. Environment Setup
Create a \.env\ in the \ackend\ folder with your MongoDB URI.
\\\env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
\\\

### 4. Run the Project
\\\ash
# In backend folder
npm run dev

# In frontend folder
npm run dev
\\\
